Files included in this table:
  • Reads - Raw reads in FASTQ format will be available upon request at dbGAP.
    Each FASTQ file was aligned without multimappers on hg19 (GRCh37 Genome Reference Consortium Human Reference 37).
    Alignment is available in BED.GZ format.
  • QC - Libraries QC information by fastqc.
  • BigWigs - visualization of reads for each donor in BigWig format.
  • Peaks - Peaks were obtained using novel semi-supervised peak calling approach.
  • Visual Labels - Handful annotation of locations as peaks, valleys or peak shores, used to obtain peaks, that is optimal for a given sample.
  • Peak Caller Models - Trained SPAN Peak Analyzer models for each sample.

Learn more about visual peak calling and find out how to apply this approach to your data in How-To tutorial.
