Modification UCSC Genome Browser
Materials included in this table:
  • Preconfigured sessions for UCSC Genome Browser.
  • Session Type - Basic or Extended.

    Basic session contains reads visualization in BigWig format, individual peaks for each sample,
    and labels used in semi-supervised peaks analysis.
    ENCODE tracks for classical CD14+ monocytes are included for control.
    Extended session contains additional peaks produced by golden standard peak callers, i.e. MACS2 and SICER.
