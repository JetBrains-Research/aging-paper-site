This page contains materials for all histone modifications processed in "Multiomics dissection of healthy human aging" experiment. @TABLE@
Modification Session Type UCSC Genome Browser Washu Epigenome Browser
Materials included in this table:
  • Session Type - Basic or Extended.
    Basic session contains reads visualization in BigWig format, individual peaks for each sample, and labels used in semi-supervised peaks analysis. Also it contains ENCODE tracks for CD14+ monocytes.
    Extended session contains additional peaks produced by golden standard peak callers, i.e. MACS and SICER.
  • UCSC Genome Browser - preconfigured sessions for UCSC Genome Browser.
  • WashU Epigenome Browser - preconfigured sessions for WashU Epigenome Browser.

Sessions

Along with preconfigured online genome browser sessions, we prepared sessions for desktop genome browsers: IGV Genome Browser and JBR Genome Browser. Click on the corresponding link to download application.
Please NOTE, JBR genome browser can be used as part of integrated solution for semi-supervised peak calling. Section Tools contains details of the method and follow the step-by-step tutorial. @TABLE2@
