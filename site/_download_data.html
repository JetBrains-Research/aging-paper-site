ChIP-Seq

@TABLE_CHIPSEQ@
Modification Alignment QC Bigwigs Peaks Visual Labels Peak Caller Models
Files included in this table:
  • Alignment - aligned reads in BED.GZ format,
    aligned without multimappers on hg19 (GRCh37 Genome Reference Consortium Human Reference 37).
    Raw reads in FASTQ format will be available upon request at dbGAP.
  • QC - library QC information by fastqc.
  • BigWigs - visualization of reads for each donor in BigWig format.
  • Peaks - BED files with peaks obtained using novel semi-supervised peak calling approach.
  • Visual Labels - manual annotations of genome locations as peaks, valleys or peak shores, used to obtain optimal peaks for a given sample.
  • Peak Caller Models - trained SPAN Peak Analyzer models for each sample.

Learn more about visual peak calling and find out how to apply this approach to your data in How-To tutorial.


DNA Methylation

Alignment QC Filtered Cytosines Methylation Level Coverage DMRs
Files included in this table:
  • Alignment - Reduced Representation Bisulfite Sequencing data,
    aligned on hg19 (GRCh37 Genome Reference Consortium Human Reference 37).
  • QC - library QC information by fastqc.
  • Filtered Cytosines - filtered cytosines in BED format, used in downstream analysis.
    Filtration criteria: coverage ≥ 3, absolute difference in methylation level ≥ 0.025.
  • Methylation Level - methylation level for each cytosine after filtration.
  • Coverage - coverage for each cytosine after filtration.
  • DMRs - Differentially Methylated Regions provided by MethPipe methylation analysis pipeline.

RNA-Seq

Alignment QC BigWigs Counts Phantasus
Files included in this table:
  • Alignment - RNA-Seq data,
    aligned on hg19 (GRCh37 Genome Reference Consortium Human Reference 37).
  • QC - library QC information by fastqc.
  • BigWigs - visualization tracks.
  • Counts - raw counts produced by DESeq2 package, used in downstream analysis.
  • RDA file, which can be uploaded to Phantasus RNA-Seq online analysis tool.
  • DMRs - Differentially Methylated Regions provided by MethPipe methylation analysis pipeline.
