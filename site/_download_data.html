ChIP-Seq

@TABLE_CHIPSEQ@
Modification Alignment QC BigWigs Peaks Visual Labels Peak Caller Models
Files included in this table:
  • Alignment - aligned reads in BED.GZ format,
    aligned without multimappers on hg19 (GRCh37 Genome Reference Consortium Human Reference 37).
    Raw reads in FASTQ format will be available upon request at dbGAP.
  • QC - library QC information by fastqc.
  • BigWigs - visualization of reads for each donor in bigWig format.
  • Peaks - BED files with peaks obtained using a novel semi-supervised peak calling approach.
  • Visual Labels - manual annotations of genome locations as peaks, valleys or peak shores, used to obtain optimal peaks for a given sample.
  • Peak Caller Models - trained SPAN Peak Analyzer models for each sample.

Learn more about visual peak calling and find out how to apply this approach the in How-To tutorial.


DNA Methylation

Methylation Calls QC Filtered Cytosines Methylation Level Coverage DMRs
Files included in this table:
  • Methylation Calls - preprocessed Reduced Representation Bisulfite Sequencing data,
    aligned on hg19 (GRCh37 Genome Reference Consortium Human Reference 37).
  • QC - summarised RRBS processing pipeline QC reports provided for each sample by the facility.
  • Filtered Cytosines - filtered cytosines in BED format, used in downstream analysis.
    Filtration criteria: mean coverage across samples ≥ 10.
  • Methylation Level - table with per donor methylation levels for remained after filtration cytosines.
  • Coverage - table with per donor coverage for remained after filtration cytosines.
  • DMRs - annotated Differentially Methylated Regions generated by the MethPipe pipeline.

Files included in this table:
